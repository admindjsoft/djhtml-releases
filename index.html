Features

Everything You Need
For Broadcast Graphics

A professional motion graphics editor built for broadcast — design, animate, and export stunning overlays with a familiar timeline-based workflow.

Keyframe Animation
Precise keyframe control for position, scale, rotation, opacity, and more. Multiple easing types including cubic bezier curves for smooth, professional motion.
Advanced Layer System
Text, shapes, solids, images, PNG/TGA sequences, crawl tickers, image loaders, atlas sprites, and nested compositions. Full transform controls with anchor point support.
Masks & Feathering
Adjustable masks with independent transforms, animated feathering for soft edges, and mask expansion for creative effects.
Rich Text Engine
Full typography controls with font families, weights, stroke, shadow, gradient fills, and animatable font size. Perfect for lower thirds and titles.
Gradient System
Linear and radial gradients on any layer type — shapes, solids, and text. Full color picker integration with angle and spread controls.
Freehand Drawing
Draw directly on compositions with configurable stroke width and colors. Strokes are layer-bound and follow transforms for seamless integration.
Crawl & Ticker
Create horizontally scrolling news tickers with adjustable speed, direction, and start position. Crawl layers loop automatically for continuous broadcast.
Nested Compositions
Embed compositions inside other compositions as layers. Build complex, reusable motion graphics elements like animated logos and recurring lower thirds.
Image Loader
Placeholder layers for dynamic image replacement during live broadcast. Swap photos and logos in real-time via the JavaScript API with fit/fill modes.
Preset Text Animations
One-click professionally designed text animations for text and crawl layers. Apply a preset, then customize keyframes and easing to fit your design.
Atlas / Sprite Sheet Import
Import sprite sheets with configurable grid layout. Specify columns, rows, frame count and frame rate to create animated layers from atlas images.

FAQ

Frequently Asked Questions

What is DJ HTML Creator?
DJ HTML Creator is a professional motion graphics application for Windows. Design animated broadcast graphics — lower thirds, tickers, overlays, and more — using a familiar timeline-based editor. It exports self-contained HTML templates that work with CasparCG, OBS Studio, vMix, and any HTML5-capable playout system.
Do I need coding knowledge?
No! DJ HTML Creator has a completely visual interface. You design your graphics by adding layers, setting keyframes, and adjusting properties — all without writing a single line of code. The application generates the HTML, CSS, and JavaScript automatically.
Which playout systems are supported?
DJ HTML Creator exports standard HTML5 templates that work with CasparCG (via its HTML producer), OBS Studio (via Browser Source), vMix (via Web Browser input), and any system that supports HTML overlays. The exported templates include a JavaScript API for controlling playback and updating dynamic text from your playout system.
Can I try before I buy?
Yes! We offer a free 7-day trial — no credit card required. The trial includes up to 2 compositions, a 30-frame limit, and crawl text limited to 30 characters. After the trial, choose between our monthly (€30/month) or annual (€300/year) plan for unlimited access.
What's included in future updates?
We're actively developing Lottie export, additional drawing tools, and expanded gradient features. All subscribers receive free updates as they're released.
